# DynWhitelist

## Description
DynWhitelist is an unofficial Minecraft mod that adds system of DynWhiteList.

## Features
- Checking if a player is in the database (or another storage), when player tries join to server 

## How to Use
1. Setup Minecraft Server **1.21.x**
2. Install **Fabric**
3. Put the mod `.jar` file into the `mods` folder
4. Launch the server
5. Read a config file `dyn-whitelist.properties`
6. `whitelist=false` in properties is required

## Configuration
- Config file: `dyn-whitelist.properties`

## Compatibility
- Minecraft: 1.21.x
- Mod Loader: Fabric
